# Project Plan: Aisyah Adjie Professional Portfolio

**Owner:** Haikal (Lead Developer)
**Subject:** Siti Aisyah Adjie
**Tech Stack:** Next.js 14/15 (App Router), Tailwind CSS, Framer Motion
**Status:** Independent Project (Separate from Pikuland)

---

## 1. Project Overview

Membangun sebuah web portofolio profesional untuk Siti Aisyah Adjie guna meningkatkan peluang diterima di posisi **Customer Service Representative** (khususnya di industri outdoor seperti Falisha Journey). Web ini mengedepankan sisi "Adventurous Professional".

---

## 2. Visual & Design Identity

- **Theme:** Earthy, Bold, and Clean.
- **Color Palette:**
  - `Primary`: #064E3B (Forest Green - Tailwind `emerald-900`)
  - `Secondary`: #F5F5F4 (Warm Stone - Tailwind `stone-100`)
  - `Accent`: #EA580C (Burnt Orange - Tailwind `orange-600`)
- **Typography:** Sans-serif (Inter/Montserrat) untuk kesan modern dan mudah dibaca.
